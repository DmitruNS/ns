Описание потоков:

Flow 1: Пользователь взаимодействует с мобильным приложением.

Flow 2: Мобильное приложение связывается с 2FA/AUTH для аутентификации.

Flow 3: 2FA/AUTH передает данные на Backend.

Flow 4: Backend общается с AML/CFT, хранилищем крипты и банковской системой.

Flow 5: Backend синхронизируется с партнерскими кассами, магазинами и криптобиржами.
