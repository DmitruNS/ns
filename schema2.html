Использование SVG позволяет нам с большей точностью изобразить сложные связи между компонентами. Вот пример HTML-кода с SVG-элементами для вашей схемы: ```html Схема проекта

Описание потоков:

1: Пользователь ↔️ Мобильное приложение

2: Мобильное приложение ↔️ 2FA / AUTH

3: 2FA / AUTH ↔️ Backend

4: Backend ↔️ Хранилище крипты

5: Backend ↔️ Банк/Платежная система

``` Этот код создает SVG-диаграмму с блоками и стрелками, которые отображают компоненты и их связи. Пожалуйста, скопируйте этот код в HTML-файл и откройте его в вашем веб-браузере.
